# ECE Night Tournament 2026

This repository contains the promotional and informational website for the ECE Night Tournament 2026. The project is designed to showcase the event in a polished, modern single-page format with engaging visuals and clear event presentation.

## Overview

The website is a static landing page built to represent the tournament experience. It includes:
- A welcoming hero section for the event
- Visual branding and logo placement
- A clean, presentation-friendly layout for visitors
- Content centered around the ECE Night Tournament 2026 theme

## Project Structure

- `index.html` — the main HTML file used for deployment
- `index (3).html` — the original source file
- `logotransparent2 (4).png` — the tournament logo image used in the design

## How to View the Project

You can view the website locally in either of the following ways:

1. Open `index.html` directly in your browser.
2. Start a simple local server from the project folder.

### Option 2: Using Python

```bash
python -m http.server 8000
```

Then open your browser and visit:

```text
http://localhost:8000
```

## Vercel Deployment

This project is ready to be deployed on Vercel as a static website. Simply connect the repository to Vercel and it will serve the root HTML file automatically.
